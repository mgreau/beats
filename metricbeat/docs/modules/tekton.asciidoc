[float]
=== Example configuration

The tekton module supports the standard configuration options that are described
in <<configuration-metricbeat>>. Here is an example configuration:

[source,yaml]
----
metricbeat.modules:
# Tekton Pipelines. To collect all Mixer-generated metrics
- module: tekton
  metricsets: ["taskruns"]
  enabled: false
  period: 10s
  # use tekton-pipelines-controller:9090, when deploying Metricbeat in a kuberentes cluster
  hosts: ["localhost:9090"]
  metrics_path: /metrics

----
